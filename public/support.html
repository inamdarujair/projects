❓ Frequently Asked Questions

🔧 How do I build a PC with PC-SS?

Simply visit the main page, set your budget, choose your purpose (gaming, office, etc.), and start selecting components. AUTORIG will automatically check compatibility and provide suggestions.

💰 Are the prices accurate?

Prices are estimates based on current market data. Actual prices may vary depending on your location and retailer. Always verify prices before purchasing.

⚡ How accurate are performance estimates?

Performance estimates are based on general benchmarks and component specifications. Actual performance may vary based on system configuration, software, and other factors.

🔍 Why is my build showing compatibility issues?

Compatibility issues can occur due to socket mismatches, insufficient power supply wattage, or memory type incompatibilities. Check the validation results for specific details.

💡 How do I get build suggestions?

Visit the Suggestions page, set your budget and purpose, then click "Generate Build" for AI-powered recommendations.

🛠️ Troubleshooting Guide

🚫 Build Validation Fails

Common Solutions:

  • Check CPU and motherboard socket compatibility
  • Ensure PSU wattage meets component requirements
  • Verify RAM type matches motherboard specifications
  • Check case form factor compatibility

🐌 Slow Performance

Optimization Tips:

  • Clear browser cache and cookies
  • Disable browser extensions temporarily
  • Try a different browser
  • Check your internet connection

💾 Data Not Saving

Solutions:

  • Enable JavaScript in your browser
  • Check if local storage is enabled
  • Try refreshing the page
  • Clear browser data and try again

🎨 Display Issues

Fix Steps:

  • Update your browser to the latest version
  • Check if hardware acceleration is enabled
  • Try disabling browser themes/extensions
  • Reset browser zoom to 100%
